This is a Python-based project developed as an introductory tool for Natural Language Processing (NLP). The program analyzes user-provided text in both English and Arabic to determine the overall sentiment (Positive, Negative, or Neutral) based on a pre-defined bilingual dictionary.

Key Features
Bilingual Support: Handles and processes both Arabic and English inputs seamlessly.

Sentiment Scoring: Uses a rule-based algorithm to assign scores to text based on emotional triggers.

Data Pre-processing: Implements string cleaning and normalization (removing punctuation and handling case sensitivity) for higher accuracy.

Real-time Reporting: Provides a detailed analysis report, including word count and identified keywords.

How It Works
The user enters a sentence or a paragraph.

The system cleans the input and tokens it into individual words.

Each word is compared against a comprehensive dictionary of positive and negative terms.

A final sentiment report is generated based on the calculated score.
